
Installation des Java-Programms "TSUtil" (seit Java 1.4):

 - Die Datei "TSUtil.zip" in ein beliebiges,leeres Verzeichnis entpacken.
 - Die Umgebungsvariable 'CLASSPATH' erzeugen/modifizieren und den Namen des Verzeichnisses angeben.
	z.B.	CLASSPATH=.;C:\TSUtil	oder	CLASSPATH=.;C:\Route_Riter\TSUtil
(Achtung: Der Wert '.' muss zusaetzlich in der Definition vorhanden sein.)

Prüfen ob Datei gefunden:
 - Dos-Eingabeumgebung öffen
 - "Java TSUtil" (Gross/Kleinschreibung!) eingeben
Wenn das Programm gefunden wird erfolgt die Ausgabe:
	"Fehler - Zu wenig Argumente!"
sowie weitere Zeilen.



Es ist auch eine jar-Datei-Version von TSUtil verfügbar, die alternativ oder zusätzlich benutzt werden kann:

Installation:
  - Die Datei "TSUtil.jar" in ein beliebiges Verzeichnis kopieren/entpacken.

Funktionsprüfung:
 - Dos-Eingabeumgebung öffen
 - das Verzeichnis mit der jar-Datei aktivieren (aktuelles Verzeichnis) 
 - "Java -jar TSUtil.jar" (Gross/Kleinschreibung!) eingeben
Wenn das Programm erfolgreich aktiviert wird erfolgt die Ausgabe:
	"Fehler - Zu wenig Argumente!"
sowie weitere Zeilen.

Installation of Program "TSUtil" (since Java 1.4):

 - decompress the file "TSUtil.zip" into a new or empty directory.
 - create or modify the environment variable 'CLASSPATH' and add the full path to the folder.
	eg.	CLASSPATH=.;C:\TSUtil	or	CLASSPATH=.;C:\Route_Riter\TSUtil
(Attention: '.' must be included within the definition.)

To check the availability of the Utility:
 - Open a Dos-Window
 - Enter: "Java TSUtil" (pay attention to case of letters)
If the Utility can be activated you will see the Message:
	"Error -- Too less arguments!"
followed by other lines.



You can also use the attached jar-File to run TSUtil:

Installation:
 - copy or decompress the file "TSUtil.jar" into an arbitrary directory

Check of availiability:
 - Open a Dos-Window
 - Branch to the directory where the jar-file is located
 - Enter: "Java -jar TSUtil.jar" (pay attention to case of letters)
If the Programm can be activated successfully you will see the Message:
	"Error -- Too less arguments!"
followed by other lines.
